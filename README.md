
# Well-Log Data Repository
This repository contains 5 well-log datasets (CSV format) used in the associated analysis/article.

## File List
- `W1.csv`: Log data for Well 1
- `W2.csv`: Log data for Well 2
- `W3.csv`: Log data for Well 3
- `W4.csv`: Log data for Well 4
- `W5.csv`: Log data for Well 5

## Data Description
Each CSV file includes standard well-log measurements (e.g., depth, resistivity, porosity, gamma ray) collected from the corresponding well. Column headers in each file define the specific log parameters.

## Usage
These datasets support the analysis, visualization, or modeling workflows described in the associated work. Load the CSV files using standard data tools (e.g., Python `pandas`, Excel, MATLAB).
